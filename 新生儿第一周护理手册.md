| 天数 | 核心任务 | 护理重点 |

## 三、重要禁忌清单（第一周特别版）

### ❌ 绝对不要做

| 禁忌行为 | 风险 | 来源依据 |
|:---|:---|:---|
| **挤乳头** | 感染、乳腺损伤 | 第1个月禁忌第5条 ✅ |
| **绑腿** | 影响髋关节发育，无科学依据 | 第1个月禁忌第2条 ✅ |
| **把屎把尿** | 脊柱损伤、脱肛 | 第2个月禁忌第2条 ✅ |
| **剃光胎毛** | 毛囊损伤、头皮感染 | 第1个月禁忌第1条 ✅ |
| **捏鼻梁** | 无助于鼻梁变挺，可能损伤鼻黏膜 | 第1个月禁忌第6条 ✅ |
| **戴手套** | 限制触觉发育、存在安全隐患 | 第1个月禁忌第8条 ✅ |
| **用力摇晃哄睡** | 婴儿摇晃综合征（脑损伤） | 第2个月禁忌第7条 ✅ |
| **使用闪光灯拍照** | 视网膜损伤 | 第2个月禁忌第8条 ✅ |

### ⚠️ 需要澄清的误区

| 原图说法 | 医学修正 | 说明 |
|:---|:---|:---|
| "侧睡防呛奶" | **仰卧才是最安全的睡姿** | 美国儿科学会(AAP)建议：健康婴儿应**仰卧**睡觉，可降低SIDS风险。喂奶后拍嗝，有吐奶风险时可侧卧但**必须有人看护** |
| "第五天黄疸高峰期，记得晒太阳" | **晒太阳退黄效果有限，且有风险** | 隔着玻璃晒无效，直接暴晒可能晒伤或着凉。**黄疸监测应以经皮测疸或血检为准**，病理性黄疸需光疗 |
| "穿衣摸后颈" | ✅ 正确 | 判断冷热应摸后颈，而非手脚（第4题） |

## 五、需补充的重要信息

以下内容是原图/文本**未提及但对第一周至关重要**：

### 🚨 危险信号（需立即就医）
- 体温 > 38℃ 或 < 36℃
- 呼吸急促（>60次/分）、呻吟、三凹征
- 持续嗜睡或异常烦躁
- 拒奶超过2次
- 抽搐
- 脐带红肿渗液、有臭味
- 皮肤发紫或苍白

### 📝 实用技巧
- **襁褓包裹**：可减少惊跳反射，帮助入睡（注意髋关节能活动）
- **肌肤接触**：每天1小时以上，稳定宝宝心率体温，促进泌乳
- **记录喂养排泄**：用手机APP记录，方便医生评估
